single_pixel_buffer_v1

  • # wp_single_pixel_buffer_manager_v1

    The wp_single_pixel_buffer_manager_v1 interface is a factory for single-pixel buffers.

    Requests

    • #
      destroy
      destroy the manager
      void wp_single_pixel_buffer_manager_v1_destroy(
      struct wp_single_pixel_buffer_manager_v1* wp_single_pixel_buffer_manager_v1
      )
      Destroy the wp_single_pixel_buffer_manager_v1 object. The child objects created via this interface are unaffected.
    • #
      create_u32_rgba_buffer
      create a 1×1 buffer from 32-bit RGBA values
      struct wl_buffer* wp_single_pixel_buffer_manager_v1_create_u32_rgba_buffer(
      struct wp_single_pixel_buffer_manager_v1* wp_single_pixel_buffer_manager_v1,
      uint32_t r, /* value of the buffer's red channel */
      uint32_t g, /* value of the buffer's green channel */
      uint32_t b, /* value of the buffer's blue channel */
      uint32_t a /* value of the buffer's alpha channel */
      )
      Create a single-pixel buffer from four 32-bit RGBA values. Unless specified in another protocol extension, the RGBA values use pre-multiplied alpha. The width and height of the buffer are 1.
